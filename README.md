# BitBurner Scripts

This repository contains scripts for the game Bitburner.

## Setup

This repository is configured to work with an in-game script that pulls files from GitHub into Bitburner.

### File List

The `files.txt` file contains a list of scripts that should be pulled into the game. Add one filename per line.

## Getting Started

1. Add your script files to this repository
2. Update `files.txt` with the filenames you want to sync
3. Use your in-game sync script to pull the files from GitHub
